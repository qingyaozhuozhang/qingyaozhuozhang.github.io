# 生活常用



### 共享文档

### 1️⃣ 第一步：登录腾讯文档

- **访问官网**：在浏览器中打开腾讯文档官网 [docs.qq.com](https://docs.qq.com)（或者直接在电脑上打开腾讯文档客户端）。
- **快捷登录**：使用你的微信或 QQ 扫码即可登录。

### 2️⃣ 第二步：导入本地 Excel 文件

- **找到入口**：登录进入主界面后，在页面右上方（或“新建”按钮旁）找到 **“导入”** 按钮。
- **选择文件**：在下拉菜单中选择 **“导入本地文件”**。
- **上传表格**：在弹出的窗口中，找到并选中你刚才生成的 `寒假学习小组记录表.xlsx` 文件，点击打开。

### 3️⃣ 第三步：确认在线编辑模式

- **等待加载**：文件上传进度条走完后，会自动在浏览器中打开。
- **检查模式**：腾讯文档通常会自动将其转换为在线表格格式。如果在顶部看到提示，请确保表格处于 **在线编辑模式**，而不是纯粹的预览模式。

### 4️⃣ 第四步：设置权限并分享给组员

> **💡 关键步骤** 为了让刘志钰、王亚飞等其他组员都能在自己的格子里插入照片和写说明，你必须开放编辑权限：

- **点击分享**：在表格界面的右上角，点击蓝色的 **“分享”** 按钮。
- **更改权限**：在弹出的分享面板中，注意看“权限设置”部分（默认往往是“只能查看”）。点击下拉箭头，将权限修改为 **“所有人可编辑”**（或者“获得链接的人可编辑”）。
- **发送链接**：修改完毕后，点击下方的 **“复制链接”**（也可以选择直接点击图标分享到微信/QQ）。
