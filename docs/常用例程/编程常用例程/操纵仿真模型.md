### 操纵仿真模型

- **移动模型**
  - 移动海龟模型
    - `ros2 run turtlesim turtlesim_node`
    - `ros2 run turtlesim turtle_teleop_key`
  - 移动差速轮盘
    - `ros2 run teleop_twist_keyboard teleop_twist_keyboard`
- **栅格地图**
  - 使用`slam_toolbox`完成建图
    - `ros2 launch slam_toolbox online_async_launch.py use_sim_time:=True`（建图）
    - `ros2 run nav2_map_server map_saver_cli -f room`（存图）
      - 生成一张`pgm`格式的图片和对应的`yaml`格式的描述文件
  - 注意事项：
    - `slam_toolbox`无法直接使用3D点云建图
    - 栅格地图更新频率不够，创建一个配置文件，在启动时指定这个文件
      - `ros2 launch slam_toolbox online_async_launch.py use_sim_time:=True slam_params_file:=src/fishbot_navigation2/config/slam_params.yaml`
  - 使用`GIMP`修图
    - `sudo apt install gimp`
    - [gimp操作教学](https://www.bilibili.com/video/BV1pr4y1H7jN/?spm_id_from=333.337.search-card.all.click&vd_source=9276a2a746c932c7f2124992c08e06f9)
    - 修完图以后，点击`File -> Export As`导出pgm图片，不需要生成新的yaml文件
      - pgm（图片）：存储的是地图的“长相”（哪里是墙，哪里是路）
      - yaml（配置）：存储的是地图的“身份信息”（分辨率、原点坐标、图片文件名等）（直接用原来的就行）
